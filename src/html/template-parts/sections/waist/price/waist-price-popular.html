Ограничения после операции

В течение реабилитационного периода женщине необходимо соблюдать строгие ограничения, связанные с физической активностью. Это обусловлено тем, что к ребрам крепятся постуральные мышцы. Они всегда включаются в работу при наклонах, сгибании и разгибании туловища. При резких движениях волокна способны потянуть за ребро и привести к его надлому. Поэтому в течение двух месяцев под запретом остаются:

@@loop( '../../blepharoplasty/price/list.html', [ { 'text': 'Занятия спортом;', 'class': '' }, { 'text': 'Продолжительная ходьба, бег;', 'class': '' }, { 'text': 'Поднятие тяжестей;', 'class': '' }, { 'text': 'Резкие наклоны туловища.', 'class': '' } ] )

Обычные бытовые нагрузки опасности не представляют. Но важно избегать малейших травм: даже небольшой удар, например, об угол стола, может привести к повторному перелому и увеличить сроки восстановления.
